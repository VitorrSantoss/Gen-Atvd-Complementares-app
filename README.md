# Gen Atividades Complementares App

Um aplicativo mobile criado para estudantes, focado em simplificar o registro, acompanhamento e gerenciamento de atividades e horas complementares acadêmicas.

> **⚠️ Status do Projeto:** Este aplicativo está atualmente em fase de desenvolvimento ativo. Estruturas, telas e funcionalidades estão sendo construídas e aprimoradas continuamente.

## 📱 Sobre o Projeto

A gestão de horas complementares costuma ser um processo manual e suscetível a perdas de documentos. O objetivo principal deste aplicativo é centralizar essas informações, permitindo que o aluno tenha controle total sobre seu progresso acadêmico de forma rápida e intuitiva, diretamente do seu smartphone.

## 🛠️ Tecnologias Utilizadas

O projeto está sendo construído com foco em performance e escalabilidade, utilizando as seguintes tecnologias:

- **React Native** (0.81.5)
- **Expo** (~54.0.33) e **Expo Router** para roteamento
- **TypeScript** para tipagem estática e manutenibilidade
- **React Navigation** para estruturação de menus e abas
- **React Native Reanimated & Gesture Handler** para interações fluidas
